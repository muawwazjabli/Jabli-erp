ڈاؤنلوڈ مالیات (Finance Report)

گوگل کلائنٹ آئی ڈی حاصل کرنے کا طریقہ

ضروری نوٹ:

گوگل ڈرائیو کلاؤڈ سینک استعمال کرنے کے لیے آپ کو اپنی Client ID بنانی ہوگی۔ یہ صرف ایک بار کا سیٹ اپ ہے۔

  1. گوگل کلاؤڈ کنسول کھولیں: console.cloud.google.com پر جائیں۔
  2. نیا پروجیکٹ بنائیں: اوپر بائیں کونے میں Select a project اور پھر New Project پر کلک کریں۔ نام رکھیں (مثلاً: POS-Backup)۔
  3. Google Drive API آن کریں: سرچ بار میں "Google Drive API" لکھ کر سرچ کریں اور اسے Enable کریں۔
  4. OAuth Consent Screen سیٹ کریں: بائیں مینو سے OAuth consent screen پر جائیں، External منتخب کریں، اور بنیادی معلومات (ایپ کا نام اور ای میل) درج کر کے سیو کریں۔
    * اہم: Test Users والے سیکشن میں اپنی ای میل (جس پر بیک اپ لینا ہے) لازمی شامل کریں۔
  5. کلائنٹ آئی ڈی بنائیں: بائیں مینو سے Credentials پر کلک کریں۔ اوپر + Create Credentials پر کلک کریں اور OAuth client ID منتخب کریں۔ Web Application منتخب کریں۔
  6. Javascript Origins: Authorized JavaScript origins میں وہ یو آر ایل شامل کریں جس پر آپ کا سافٹ وئیر چل رہا ہے (مثلاً: http://localhost:5500)۔
    * نوٹ: اگر آپ فائل براہ راست کھول رہے ہیں (file:///) تو گوگل اسے بلاک کر دے گا۔ اسے کسی لوکل سرور (Live Server) پر چلائیں۔
  7. آئی ڈی کاپی کریں: آخر میں ملنے والی Client ID کاپی کریں اور سافٹ وئیر کی سیٹنگز میں پیسٹ کر دیں۔

Error 400 (Access Blocked) کا حل:

  • یقینی بنائیں کہ آپ کا سائن ان ای میل Test Users کی لسٹ میں موجود ہے۔
  • گوگل کونسول میں Authorized JavaScript origins میں وہی لنک ڈالیں جو آپ کے براؤزر میں نظر آرہا ہے۔
  • ایپ کو براؤزر میں localhost کے ذریعے اوپن کریں، ڈائریکٹ فائل اوپن نہ کریں۔
